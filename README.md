# Fmetal

🔸 Web site for small business... <br>
🔸 This web app built with using [React](https://reactjs.org/).

## 💻 Setup <br>

### Install Dependencies

```
npm install
```

### Run the App in the Dev Server

```
npm start
```

### to Build for Production

```
npm build
```

## 🔥 Technologies and Libraries <br>

✔️ React.js <br>
✔️ react-icons <br>
✔️ react-scroll <br>
✔️ animate.css <br>
